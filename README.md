## 📖 프로젝트 소개

한국의 각종 전자책 플렛폼들의 독서노트를 Readwise로 내보내는 크롬 확장 프로그램 입니다.

### 지원 플렛폼

- [x] 리디북스
- [ ] 알라딘
- [ ] 교보문고
- [ ] Yes24
- [ ] 밀리의 서재

## 🚀 시작하기

[크롬 웹 스토어](https://example.com)에서 바로 다운받아 사용하실 수 있습니다.

### 개발

```bash
pnpm install

pnpm run dev
# chrome://extensions -> (ON: Developer mode) -> Load unpacked -> <프로젝트 폴더/dist>  선택
```
